# Bible Voice Finder

A simple tool to interpret (voice / speech) input describing a Bible reference (book, chapter, verse) and map it to a canonical reference in the Bible.  

This can be useful for voice assistants, speech-to-text interfaces, or systems that allow users to speak Bible references.

---

## Features

- Parse spoken Bible references (book, chapter, verses) into a structured canonical format  
- Support bridging ambiguous numeric phrases into valid references  
- Modular code: separate parsing, reference resolution, book lookup, etc.
